The Vanity Store

For my UI/UX class at Brown, I was tasked to create a ReactJS Web App for an imagined E-commerce site of my choice. The goal of the task was to create a page such that the user could:

  • Filter and Sort on item properties
  • Add/Remove Items from the cart and see aggregate cart total

This project allowed me to learn about making easy to use, interactive React Web Apps for E-commerce sites and beyond.

  • Type: ReactJS Web App

  • Timeline: December 2020

Visual Design Choices

When making the webpage, I decided to focus on usablilty

  • I used buttons with a recognisable button visual (Material UI button) that changed colour on hover. This makes it easy for the user to see what actions are avaible to them.
  • I also used tags (Material UI Chips) to clearly highlight properties for each item that could be filtered or sorted on. While, this made the cards bigger and visually busy, it was important to highlight this information in a relatively concise manner
  • In terms of information heirarchy, I kept the sorting and filtering actions at the top so the user sees the actions clearly. I also kept the cart and items side by side so that the user browses through items with the subtle reminder to add to their cart.

Code Design Choices

  • Resuable React Components: Mirror Item compoenent for browsing and cart
  • Component Structure: App, Cart, FilteredList
    • App contained Cart and Filtered List. Therefore, App was used to keep track of items before filtering, after filtering and after adding/removing from cart.
    • FilteredList contained the filtering logic and rendering using information from App.jsx and user input.
    • Cart contained all the cart logic and rendering using user input with information from FilteredList

Final Thoughts

Through this task, I got a lot of value from learning about how the React DOM works as well as using state and JSX Components. Once I got the hang of using React, I was suprised how easy it was to use Material-UI React Framework and ensure usability. I look forward to using React in the future!
